My Agent-Based Model

This agent-based model simulates a number of agents moving around and consuming resources from their environment. The agents move around more quickly if they have fewer resources in their stores and they also share their resources with other nearby agents.
The model produces an animation of the agents eating the environment, and can be run from the command line.

To run the model:

  • Download the relevant files using the link below.
  • Open the Command Prompt / Terminal.
  • Set the directory to the place where you have saved the model files.
  • Type into the Command Prompt / Terminal: "python animationmodel.py 10 100 20" and press Enter.

The three numbers represent, in order, the number of agents, the number of iterations, and the size of the neighbourhood (for interacting with nearby agents). You can re-run the model with different parameters by simply typing in different numbers.
